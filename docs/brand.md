# ReActor brand system

ReActor is controlled intelligence for builders: an agent that can act with
momentum while keeping the human in control. The identity is built around a
geometric `R` and a returning loop. The `R` is the actor; the loop is the
reaction; the open path is the explicit pause point that keeps autonomy bounded.

Use the mark alone at 16–48 px. Use the wordmark when the product name is
introduced. Keep the loop visible; do not crop the arrow or redraw the mark in
a different gradient.

## Palette

| Token | Hex | Use |
| --- | --- | --- |
| Reactor Black | `#070B0D` | application canvas and terminal surfaces |
| Ion Lime | `#D8FF4F` | primary action, active state, mark start |
| Reactor Teal | `#54D7C7` | return path, links, secondary signal |
| Warm Signal | `#F3F5EE` | primary text and light surfaces |
| Field Gray | `#9CAAA5` | supporting text, labels, metadata |

The palette is intentionally narrow. Lime means “act”; teal means “return”;
warm signal is the readable human layer.

## Typography and voice

Use a clean grotesk/system sans for product names and UI, with a restrained
monospace for commands, telemetry, and technical labels. Headlines should be
short and direct. The product voice is calm, precise, and operational:

> Make the next move.

Avoid generic AI language, neon rainbow gradients, decorative sparkles, and
unbounded claims about autonomy.

## Product surfaces covered

The identity is applied to the repository hero and icon, collaboration web
favicons and social card, worker icon assets, stats navigation and chart
palette, and the collaboration manifest/theme colors. New surfaces should use
the same tokens and reference the source SVGs rather than introducing a second
logo.
